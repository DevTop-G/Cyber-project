# 📋 AegisAI System Quick Reference\n\n## 🎯 System Overview\n\nAegisAI is a **Multi-Agent AI Threat Detection System** that combines:\n- **4 Specialized AI Agents** (External, Content, Synthesizer, Prompt Injection)\n- **6+ Machine Learning Algorithms** (SVM, MNB, LR, DeBERTa, DistilBERT, MiniLM)\n- **Legacy ML + Modern Transformers** (Hybrid approach)\n- **Inter-Agent Communication** (Consensus voting)\n\n---\n\n## 🔍 The 4-Agent Architecture\n\n### **Agent 1: External Analysis** 🌐\n```\nINPUT:  URLs, Domains\nALGOS:  SVM + TF-IDF | Embeddings | Pattern Matching | SequenceMatcher\nOUTPUT: url_risk, url_ml_risk, domain_similarity, risk_factors\nRUNTIME: ~50ms\n```\n\n### **Agent 2: Content Analysis** 📧\n```\nINPUT:  Email Text\nALGOS:  MNB + TF-IDF | DeBERTa | DistilBERT | MiniLM | Pattern Matching\nOUTPUT: phishing_prob, spam_prob, sentiment, connection_score, keywords\nRUNTIME: ~150ms  \n```\n\n### **Agent 3: Synthesizer** 🤖\n```\nINPUT:  Results from Agents 1, 2, 4\nALGO:   Weighted Ensemble Voting\nOUTPUT: risk_score, risk_level, threat_types, forensic_explanation\nRUNTIME: ~5ms\n```\n\n### **Agent 4: Prompt Injection** 🔐\n```\nINPUT:  Original Text\nALGOS:  DeBERTa MNLI | Regex Patterns (20+ rules)\nOUTPUT: prompt_injection_detected, confidence, attack_categories\nRUNTIME: ~20ms\n```\n\n---\n\n## 🧠 Legacy ML Algorithms (Training Data in models/ folder)\n\n### **Multinomial Naive Bayes (MNB)** 📊\n- **File**: `model_new.pkl`\n- **Data**: phishing_email_dataset.csv + spam_detection_dataset.csv\n- **Speed**: ⚡⚡⚡ Very Fast\n- **Accuracy**: 75-80%\n- **Use**: Baseline text classification (Agent 2)\n\n### **Support Vector Machine (SVM)** 🎯\n- **File**: `phishing_new.pkl`  \n- **Data**: malicious_url_dataset.csv\n- **Speed**: ⚡⚡ Medium\n- **Accuracy**: 82-88%\n- **Use**: URL classification (Agent 1)\n\n### **Logistic Regression** 📈\n- **Embedded**: In scikit-learn pipeline\n- **Data**: Same as MNB\n- **Speed**: ⚡⚡⚡ Fast\n- **Accuracy**: 70-75%\n- **Use**: Validation baseline (Agent 2)\n\n### **TF-IDF Vectorization** ✂️\n- **Files**: `vectorizer_new.pkl` (text), `vectorizerurl_new.pkl` (URLs)\n- **Purpose**: Convert text/URLs to numerical feature vectors\n- **Used By**: MNB, SVM, Logistic Regression\n\n---\n\n## 🤖 Modern Transformers (HuggingFace)\n\n### **DeBERTa-v3-Small** (General Purpose)\n- **Model**: `microsoft/deberta-v3-small`\n- **Speed**: ⚡ Slower (80ms)\n- **Accuracy**: 85-92%\n- **Used**: \n  - Agent 2: Content phishing/spam detection\n  - Agent 4: Prompt injection (MNLI variant)\n- **Advantage**: Captures semantic nuances\n\n### **DistilBERT** (Sentiment)\n- **Model**: `distilbert/distilbert-base-uncased-finetuned-sst-2-english`\n- **Speed**: ⚡⚡ Medium (30ms)\n- **Accuracy**: 80-85%\n- **Used**: Agent 2 - Detect aggressive/urgent tone\n- **Advantage**: Efficient BERT distillation\n\n### **MiniLM-L6-v2** (Embeddings)\n- **Model**: `sentence-transformers/all-MiniLM-L6-v2`\n- **Speed**: ⚡⚡ Medium (20ms)\n- **Dimensions**: 384\n- **Used**: \n  - Agent 1: Domain similarity detection\n  - Agent 2: Email text vs URL semantic alignment\n- **Advantage**: Fast, accurate semantic matching\n\n### **DeBERTa MNLI Fine-tuned** (Prompt Injection)\n- **Model**: `mrm8488/deberta-v3-small-finetuned-mnli`\n- **Task**: Natural Language Inference (Textual Entailment)\n- **Speed**: ⚡ Slower (20ms)\n- **Used**: Agent 4 - Detect prompt injection via contradiction\n- **Key Feature**: Analyzes semantic contradictions\n\n---\n\n## 📊 Data Flow\n\n```\nUser Input (Email/URL)\n    ↓\nPreprocessor (Clean, Extract URLs)\n    ↓\n┌───┴────┬─────────┬───────────┐\n↓        ↓         ↓           ↓\nAgent1   Agent2    Agent4      (Parallel)\nURLs     Content   Injection\n    ↓        ↓         ↓\n    └────┬───┴────┬────┘\n         ↓        \n      Agent3 (Synthesize)\n         ↓\n   Final Risk Report\n         ↓\n   Next.js Frontend\n```\n\n---\n\n## ⚖️ Risk Scoring Formula (Agent 3)\n\n```\nRisk Score = \n  (url_risk × 0.30) +              # URL/Domain (30%)\n  (phishing × 0.40) +              # Content Phishing (40%)\n  (spam × 0.15) +                  # Spam Detection (15%)\n  (ai_generated × 0.10) +          # AI-Generated Content (10%)\n  (domain_similarity × 0.05) +     # Brand Spoofing (5%)\n  (injection × 0.30) +             # Prompt Injection (30% when detected)\n  divergence_penalty +              # Text ≠ URL Divergence\n  sentiment_aggression              # Aggressive Tone\n  \nRisk Level Classification:\n  MINIMAL:  < 0.22\n  LOW:      0.22 - 0.50\n  MEDIUM:   0.50 - 0.80\n  HIGH:     ≥ 0.80\n```\n\n---\n\n## 🔄 Algorithm Selection by Task\n\n| Task | Primary Algorithm | Backup | Runtime |\n|------|------------------|--------|----------|\n|
